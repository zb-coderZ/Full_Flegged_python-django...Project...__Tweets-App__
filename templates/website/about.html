About TweetApp

TweetApp is a Django-based CRUD project inspired by Twitter/X. It helps users create, edit, delete, and manage posts with authentication and secure routing.

Our Mission

To build a clean and simple social posting system using Django where users can learn real-world backend development including authentication, ORM, and CRUD operations.

What You Learn Here

  • ✔ Django Models & ORM
  • ✔ CRUD Operations (Create, Read, Update, Delete)
  • ✔ User Authentication System
  • ✔ Route Protection (Login Required)
  • ✔ Template Inheritance

Tech Stack

Django Python Tailwind CSS SQLite
